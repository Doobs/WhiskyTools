# Whisky & BBQ Utility App

A Blazor WebAssembly Progressive Web App (PWA) providing practical tools for whisky dilution and barbecue temperature conversion.  
Built with a whisky-inspired theme, custom icon, and mobile-friendly layout for a warm, polished user experience.

---

## 🚀 Features
- **Whisky dilution calculator** – precise ratios for adjusting ABV.
- **Barbecue temperature converter** – quick conversions between °C and °F.
- **Tabbed interface** – switch seamlessly between utilities.
- **PWA support** – installable on desktop and mobile.
- **Custom loader & icon** – whisky-inspired branding.
- **Responsive design** – optimized for mobile and desktop.
